# RedLine-Stealer-1-
La regla utiliza dos cadenas (strings) que se han observado en versiones anteriores de Redline Stealer, incluyendo "RedLineStealer" y "RLST". 
Si cualquiera de estas cadenas está presente en un archivo o proceso en el sistema analizado, la regla se activará y se considerará que se ha encontrado Redline Stealer.
